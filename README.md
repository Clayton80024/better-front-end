# Installo Landing Page

Modern landing page for Installo — Case intake, rebuilt for paralegals.

## Deploy on Vercel

1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **Add New** → **Project**
3. Import `Clayton80024/better-front-end`
4. Click **Deploy**

Your site will be live at `https://your-project.vercel.app`

Optional: Add `NEXT_PUBLIC_GA_MEASUREMENT_ID` environment variable for Google Analytics.
